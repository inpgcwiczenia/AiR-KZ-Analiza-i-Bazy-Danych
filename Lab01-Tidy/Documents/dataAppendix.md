# Data appendix
All files from /AnalysisData/ are described here.

##  tb.csv
This file contains data on tuberculosis among different patient groups.
There are too many columns and empty spaces.
\
Variables:
- iso2 - categorical variable; country of the patient. 
- year - categorical variable; year of examination.
- other columns - data on type of tuberculosis

## tb_tidy.csv
This file contains cumulated information from tb.csv."Other columns" have been cumulated
to display a simpler number of "cases" column, similiar to new "age" column.
